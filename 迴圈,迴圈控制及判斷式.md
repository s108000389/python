### 迴圈控制
```
break：強制跳出 ❮整個❯ 迴圈
continue：強制跳出 ❮本次❯ 迴圈，繼續進入下一圈
pass：不做任何事情，所有的程式都將繼續
```

### if
```python
if (x==1):
    print("1")
elif (x==2:
    print("2")
else:
    print("3")
```

### for
```python
for i in range(1,10,2):  #從1開始 直到10 一次加2
  print(i)
```

### while
```python
while (i != 10):    #達成條件時一直做
    i +=1
    print(i)
```
